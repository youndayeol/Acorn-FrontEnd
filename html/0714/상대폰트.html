가변 글꼴 : 글자 크기 (em) = 글자 크기 (px) / 16px

px로 표시한 글자의 크기를 16px로 나누면 em값을 계산할 수 있다

글자 지정 단위 종류 1em = 16px = 12pt와 같다

em : 부모에서 정한 글꼴의 대문자 M 너비 기준 (1em)으로 한 후 비율 값 지정

rem : 문서 시작 부분(root)에서 지정한 크기 기준 (1rem)우로 한 후 비율 값 지정

ex : 해당 글꼴의 소문난 x의 높이를 기준(1PX)으로 한 후 비율 값 지정

px : 모니터의 2픽셀 기준으로 할 후 비율 값 지정

pt : 포인트라 하며,일반 문서에서 많이 사용
